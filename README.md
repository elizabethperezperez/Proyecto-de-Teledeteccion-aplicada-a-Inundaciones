# Proyecto-de-Teledetección-aplicada-a-Inundaciones
En este proyecto se han aplicado técnicas de teledetección y procesamiento de imágenes (filtros, mascaras...) para el estudio de inundaciones, turbidez del agua y temperaturas en Venecia

El proyecto se constituye de 3 Notebooks, cada uno estudia un aspecto distinto:
- **Trabajo-Part1.ipynb** Trata de el estudio de inundaciones.
- **Trabajo-Part2.ipynb** Trata de el estudio de la turbidez del agua en Venecia.
- **Trabajo-Part3.ipynb** Trata de el estudio de la temperatura en Venecia.
